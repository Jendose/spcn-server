Для того, чтобы развернуть проект, необходимо:
1. Установить Java не ниже 11 версии
2. Установить PostgreSQL не ниже 11 версии
3. Установить Gradle не ниже версии 7.0.2
4. Открыть все модули сервиса и эмулятор в отдельных проектах IntelliJ IDEA
5. Запустить автоматическую сборку каждого из проектов
5. Запустить каждый из модулей
